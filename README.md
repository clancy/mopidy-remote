# Mopidy Remote

## Development

* Download source code  ```git clone https://clancywhittle@bitbucket.org/clancywhittle/mopidyremote.git```
* Navigate to base directory ```cd mopidyremote```
* Install dependencies ```npm install```
* Run ios ```react-native run-ios```
